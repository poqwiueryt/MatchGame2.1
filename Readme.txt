Projeto 01: Match Game


Objetivo:
        Este projeto tem como objetivo implementar um jogo de mecânica simples, porém divertida, para desenvolver conceitos básicos da linguagem C# em conjunto com a linguagem XAML para gerar a interface do programa, utilizando a plataforma Visual Studio 2019.


Mecânica de jogo:
        O jogador deve clicar em uma das figuras exibidas na janela e em seguida selecionar a figura que forme um par idêntico à primeira imagem. Se o par for formado corretamente, ambas as imagens são eliminadas da tela, caso o par não seja formado ambas as figuras permanecem. 
Assim que o número máximo de pares for encontrado, neste caso 8, pois são 16 figuras, o jogo é encerrado e seu tempo de conclusão é exibido na tela ao lado da mensagem que indica reiniciar o jogo. Seu tempo de conclusão é anotado, e caso inicie novas jogadas, ele será exibido no canto inferior direito da janela mostrando seu melhor tempo de conclusão. Sendo atualizado toda vez que concluir o jogo de forma mais rápida.
O desafio do jogo consiste em obter cliques precisos na janela e encontrar os pares o quanto antes, e, em um nível mais avançado, também é preciso levar em conta o menor caminho feito pelo seu cursor para minimizar perdas de tempo ao clicar entre os pares.


Execução do jogo:
        Para inicializar o jogo em seu computador é necessário possuir o sistema operacional Windows 10 e ter instalado a versão 5.0 do Microsoft .NET SDK.
        Caso não possua o .NET 5.0, basta se dirigir ao link (https://dotnet.microsoft.com/download/dotnet/5.0), baixar e instalar a versão compatível com seu Windows.
        Caso tenha o .NET instalado, basta abrir a pasta “Exe” e dar duplo clique sobre o arquivo MatchGame2.1 e o jogo inicializará.
